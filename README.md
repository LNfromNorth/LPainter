# LPainter

## introduce

> this is a lib to show you list, queue, stack or other link structures
> in terminal, this while help you to show the struct in a more direct
> way like this

- list
```
+-----+-----+-----+
|  1  |  2  |  3  |
+-----+-----+-----+
```

- queue
```
+-------+-----+-----+-----+-------+
|  -->  |  1  |  2  |  3  |  -->  |
+-------+-----+-----+-----+-------+
```
- stack
```
+-------+-----+-----+-----+
|  top  |  1  |  2  |  3  |
+-------+-----+-----+-----+
```
